# 一个简单但是完整的策略

def initilize():
  g.security="'000001'"
  g.start_date="'2014-01-01'"
  g.end_date="'2015-01-01'"
  set_benchmark("'000001'")
  g.cash=10000

def handle_data():
  security=g.security
  close_data=attribute_history(security, 5, 3)
  MA=close_data.mean()
  current_price=close_data[-1]
  cash = g.cash
  if current_price>MA*1.01:
    order_value(security, cash)
  else:
    order_target(security, 0)

# 策略里必须得有的两个函数

# initilize()

  • 用来初始化策略的一些信息，例如：股票池，回测的开始与结束的时间等。
  • 对策略的一些基本属性进行设定，这里使用的是全局变量g来进行设置，security为策略面向的股票，start_date为回测的开始日期，end_date为回测的结束日期，cash为用户一开始所拥有的资金。

# handle_data()

  • 在这个函数里，你需要编写你的策略，比如上面的一个例子就是一个单均线的策略。
  • 策略的制定，用来判断什么情况下需要购入或是抛售股票。
