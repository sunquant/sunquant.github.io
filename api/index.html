        # API文档

        # 基础API

        策略里必须得有的两个函数：initilize()和handle_data()。

        # initilize()

        • 用来初始化策略的一些信息，例如：股票池，回测的开始与结束的时间等。
        • 对策略的一些基本属性进行设定，这里使用的是全局变量g来进行设置，security为策略面向的股票，start_date为回测的开始日期，end_date为回测的结束日期，cash为用户一开始所拥有的资金。

        # handle_data()

        • 在这个函数里，你需要编写你的策略，比如上面的一个例子就是一个单均线的策略。
        • 策略的制定，用来判断什么情况下需要购入或是抛售股票。

        # order_value(security, amount)

        用来对某支股票进行下订单，第一个参数为进行下订单股票的代号，第二个参数为订单购入数量。

        参数 类型 是否必须 说明
        security string 是 股票代码
        amount int 是 购入数量

        # order_target(security, target)

        用来控制某支股票到额定的价格，从而实现控股，第一个参数为需要进行操作的股票的代号，第二个参数为控制该股票到达的数量。

        参数 类型 是否必须 说明
        security string 是 股票代码
        target int 是 购入目标

        # attribute_history(security, day, data_type)

        用来获取当天某支股票的历史数据，第一个参数为获取股票的代号，第二个参数为获取前多少天的数据，第三个参数为获取股票的信息。

        参数 类型 是否必须 说明
        security string 是 股票代码
        day int 是 提前天数
        data_type list 是 获取类型
